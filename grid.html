Varför?

Enklare att strukturera layout.

Enhetligt utseende.

Kompakt, redundansfri och strukturerad css.

Semantik.

Hur?

Definiera upp antal kolumner, bredden av en kolumn och bredden för rännstenen.

Element delas upp till att täcka ett visst antal kolumner

Standardbredd för ett system är 940/960px, målgruppsberoende.

Vanligast med 12,16 eller 24 kolumner i ett grid system.

Definiera bara de kolumner du ska använda, men planera för fler.

Typer av gridar.

Container baserad

Total bredd = (colwidth + gutterwidth) * #cols.

Semantisk.

Extremt flexibel utan onödig markup.

Kan inte nästla containers i spans (kräver väldigt bökig css).

Row baserad

Total bredd = colwidth * #cols + gutterwidth * (#cols -1).

Tillåter oändlig nästling.

Mycket markup för annat än simpla layouter.

Diskutabel semantik.

Responsiv grid

Antalet kolumner ändras aldrig.

Inga media-queries för att uppnå desktop layout.

768 - 979 tablet.
