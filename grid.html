Varför?

Enklare att strukturera layout.

Enhetligt utseende.

Kompakt, redundansfri och strukturerad css.

Semantik.

Hur?

Definiera upp antal kolumner, bredden av en kolumn och bredden för rännstenen.

Element delas upp till att täcka ett visst antal kolumner

Standardbredd för ett system är 940/960px, målgruppsberoende.

Vanligast med 12,16 eller 24 kolumner i ett grid system.

Definiera bara de kolumner du ska använda, men planera för fler.

Typer av gridar.

Container baserad

Totalbredd = (colwidth + gutterwidth) * #cols.

Semantisk.

Extremt flexibel utan onödig markup.

Kan inte nästla containers i spans (kräver väldigt bökig css).

Row baserad

Totalbredd = colwidth * #cols + gutterwidth * (#cols -1).

Tillåter oändlig nästling.

Mycket markup för annat än simpla layouter.

Diskutabel semantik.

Flytande grid

Anpassar sig alltid till föräldern.

Väldigt lik container griden.

Perfekt för webbappar.

Kan nästlas hur långt som helst.

Kan vara svårt att förutse layouten.

Komplexa layouter blir lätt ett vertikalt virrvarr.

Inte helt simpel att räkna ut.

Implementation

En ganska vanlig layout med navigering och någon form av layout-wide header, en meny på 4 kolumner till vänster, 16 kolumner centrerat content, 4 kolumner information till höger och en footer under. Baserat på 24 kolumner med en kolumnbredd av 22px och en gutter på 18px, detta ger en total content bredd på 942px.

Responsiv grid

Antalet kolumner ändras aldrig.

Inga media-queries för att uppnå desktop layout.

> 1200 large.

768 - 979 tablet.

<= 767 horizontell telefon, edge case.

<= 479 telefon.

Hur?

Bestäm nya storlekar för gutter och kolumner.

Tänk på totalbredden.

Vilka special element ska ändras vid vissa storlekar?

Tänk på selektorer och kombinatorer.

Kod

Large

En kolumn är nu 26px med en gutter 24px

@media (min-width:1200px)
{
	.container {width:1200px;}

	.span24,.span16,.span4 {margin:5px 12px;}

	.container > .span24{width:1176px;}
	.container > .span16{width:776px;}
	.container > .span4{width:176px;}
}

Tablet

En kolumn är nu 20px med en gutter 12px

@media (min-width:768px) and (max-width: 979px)
{
	.container {width:756px;}

	.span24,.span16,.span4 {margin:5px 6px;}

	.container > .span24{width:756px;}
	.container > .span16{width:500px;}
	.container > .span4{width:116px;}
}

Flytande

Alla kolumner är nu flytande, kan finnas specialbehov.

@media (max-width:767px)
{
	body {padding:0 12px;}

	.container,
	.container > .span24,
	.container > .span16,
	.container > .span4{width:100%;float:none;margin:0;}
}
